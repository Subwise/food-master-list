GitHub Pages setup
1
Copy the code and save it as index.html on your computer
2
Go to github.com → New repository → name it e.g. shopping-list, set to Public
3
Upload index.html via the "Add file" button in the repo
4
Go to Settings → Pages → Source: Deploy from branch → Branch: main → Save
5
Your URL will be yourusername.github.io/shopping-list — open in Safari on your phone
6
In Safari: tap the Share icon → Add to Home Screen → done, it works like an app
